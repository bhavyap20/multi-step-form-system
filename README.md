# Multi-Step Form System

## Overview

A responsive multi-step student registration form built using HTML, CSS, and JavaScript.

## Features

* 3-step registration process
* Next and Previous navigation
* Input validation
* Review & Submit page
* Progress bar
* LocalStorage support (data remains after refresh)
* Responsive and modern UI

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Open the project in VS Code.
2. Install the Live Server extension.
3. Open `index.html`.
4. Click "Open with Live Server".

## Project Structure

* index.html
* style.css
